Collage creator
===============

Homework as described in exercise.pdf. See template.png for possible input and
screenshot.png for what the program makes of it.

Build
-----

export JAVA_HOME=/usr/lib/jvm/java-6-openjdk-amd64/ # or similar, if needed
ant

Running
-------
In the build/ directory:

  1. Start the RMI registry
  2. Start server.jar
  3. Start client.jar
